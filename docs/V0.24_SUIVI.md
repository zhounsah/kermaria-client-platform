### 2.a Matrice secrets

Pour chaque secret, verifier les 9 controles :

- G : jamais commit dans Git (`git grep`)
- P : `.env.example` = placeholder uniquement
- A : jamais serialise dans reponse API
- N : jamais renvoye au navigateur (bundle client)
- L : jamais ecrit dans log applicatif (`Select-String`)
- U : jamais ecrit dans audit
- M : cache memoire acceptable (documente)
- R : rotation testee en staging
- D : rotation documentee dans SECRET_ROTATION.md

Statut :

- `[x]` control passe
- `[!]` control echoue (defect a traiter)
